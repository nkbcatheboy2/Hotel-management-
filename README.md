# 🏨 NKB Hotelier - Premium Hotel HTML Template

A modern, fully responsive, and beautifully designed frontend HTML template for a hotel business. This repository contains the **About Page** and related frontend components, designed to help hotel owners build a professional online presence, attract visitors, and manage booking inquiries seamlessly.

## ✨ Features
* **Modern UI/UX:** Clean, elegant, and professional design tailored for the hospitality industry.
* **Fully Responsive:** Mobile-first approach using **Bootstrap 5**, ensuring it looks perfect on desktops, tablets, and smartphones.
* **Integrated Booking Form:** A functional layout for checking room availability (Check-in, Check-out, Adult/Child selection) using `tempusdominus` date-pickers.
* **Interactive Elements:** Features CSS/JS animations (using `Animate.css` & `WOW.js`), animated number counters (`CounterUp`), and hover effects.
* **Rich Components:** Includes an Image Grid (About section), Team/Staff showcase, Newsletter subscription, and a comprehensive Footer.
* **Font Awesome & Bootstrap Icons:** Scalable vector icons for social links and services.

## 🛠️ Technologies & Libraries Used
* **Markup/Styling:** HTML5, CSS3
* **Framework:** Bootstrap 5
* **JavaScript:** Vanilla JS & jQuery (v3.4.1)
* **Fonts:** Google Fonts (Heebo & Montserrat)
* **Icons:** FontAwesome 5 & Bootstrap Icons
* **Plugins/Libraries:** 
  * [WOW.js](https://wowjs.uk/) (Scroll Animations)
  * [OwlCarousel](https://owlcarousel2.github.io/OwlCarousel2/) (Sliders)
  * [Tempus Dominus](https://getdatepicker.com/) (Date & Time Picker)
  * [Waypoints](http://imakewebthings.com/waypoints/) & CounterUp

## 📂 Page Structure (About Page)
1. **Header & Navbar:** Contact info, social links, and a collapsible, sticky navigation bar.
2. **Hero/Breadcrumb Section:** Stylish page header with background imagery.
3. **Room Booking Module:** Quick access form for users to enter their dates and guests.
4. **About Us Section:** Detailed text, animated statistics (Rooms, Staff, Clients), and a dynamic image grid.
5. **Our Team:** Showcase of hotel staff with individual social media links.
6. **Newsletter:** Subscription form to capture user emails.
7. **Footer:** Quick links, contact information, and copyright details.
